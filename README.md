# decimal-niner
Provides ATC audio communications for Traffic Global (JustFlight) AI aircraft in X-Plane 12 (Laminar Research) flight simulator.

This application has no affinity or relationship with JustFlight or Laminar Research.

Dependencies:
    X-Plane 12 version 12.4.4+
    Traffic Global for X-Plane 12 version TBD
    Sox
    Piper

Troubleshooting:
    You must have a radio tuned to a frequency in order to hear anything.
    Search decimal-niner output for "error"

Common Issues:
    For sox error "no default audio device"  on Windows, set environment variable AUDIODRIVER to waveaudio i.e. set AUDIODRIVER=waveaudio
